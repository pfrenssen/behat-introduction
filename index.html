What is Behat?

Behat is a BDD framework for PHP

What is BDD?

Bridge between client and development team

How does it look?

            
              Feature: Event calendar
                In order to announce upcoming events
                As an event manager
                I need to maintain a public event calendar

              Scenario: Create event
                Given I am logged in as an event manager
                When I visit the dashboard
                And I click on 'Add event'
                And I enter the title 'Summer training'
                And I enter the date '2015-08-01'
                And I click on 'Save'
                Then I should see 'Event has been created'

Gherkin

Gherkin

A language understandable by everyone

  • Business readable
  • Developer readable
  • Machine readable
  • Domain specific

Gherkin example

            
              Feature: Product highlights
                In order to promote specific products
                As a marketing manager
                I need to highlight products on the homepage

              Scenario: View product highlights
                Given I have 3 product highlights
                When I visit the frontpage
                Then I should see 3 highlighted products

Gherkin structure

            
              Feature: < title >
                In order to < business objective >
                As a < user role or persona >
                I need to < some action >

              Scenario:
                Given < a precondition >
                When < some action >
                And < some other action >
                Then < a testable outcome >
                But < something else we can test >

Scenario outlines

            
              Scenario: Show number of likes
                Given I am logged in as <user>
                When I visit the profile of <friend>
                And I click on "Like"
                Then I should see <number> likes

                Examples:
                | user   | friend | number |
                | Cindy  | Cindy  | 0      |
                | Thomas | Cindy  | 1      |
                | Thomas | Cindy  | 1      |
                | Roger  | Cindy  | 2      |
                | Roger  | Thomas | 1      |

Installing Behat

composer.json

            

              {
                "require": {
                  "behat/behat": "3.0.*@stable",
                  "drupal/drupal-extension": "~3.0"
                },
                "minimum-stability": "stable",
                "config": {
                  "bin-dir": "bin/"
                }
              }
            

          
            

              $ composer install

behat.yml

            

              default:
                suites:
                  default:
                    contexts:
                      - FeatureContext
                      - Drupal\DrupalExtension\Context\DrupalContext
                      - Drupal\DrupalExtension\Context\MessageContext
                      - Drupal\DrupalExtension\Context\MinkContext
                extensions:
                  Behat\MinkExtension:
                    goutte: ~
                  Drupal\DrupalExtension:
                    blackbox: ~
                    api_driver: 'drupal'

            When I press the :button button

            Then I should not see the link :link

Example custom steps

            

    /**
     * @Then I should see :number blog posts
     */
    public function assertBlogPostCount($number) {
      $this->assertSession()->elementsCount('css', 'div.blog', $number);
    }
